# Week 5: DOM Manipulation 

This project demonstrates the Document Object Model (DOM) using JavaScript. 

## Key Features Completed:
* **Element Selection:** Used `getElementById` and `querySelector` to target HTML elements.
* **DOM Traversing:** Navigated through parents, children, and siblings.
* **Content Modification:** Dynamically updated text, HTML, and styles.
* **Event Handling:** Implemented a functional To-Do list using form submissions.
* **Event Delegation:** Efficiently managed task deletion and completion toggles.
